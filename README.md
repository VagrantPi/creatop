# 敏捷開發實踐手冊

本電子書是敏捷軟體開發實務的培訓課程講義。

## 取得電子書

http://creatop.readbook.tw/

## 課程進行方式

Workshop（工作坊）

由講師與學員共同協作進行參與式學習

## 時間規劃

第一期共計 36 小時（每個月安排 4 週、每週 3 小時的課程）

## 主要內容規劃

* 版本控制基礎觀念
* Git 基礎指令操作教學
* Git Server 使用教學（以 GitHub 平台為主）
* AWS 雲端運算基礎教學
* Agile 敏捷軟體開發入門（採用 Jenkins）



此電子書由創科資訊技術顧問團隊製作發行，授權**創造力互動媒體**於企業內部教育訓練使用。
